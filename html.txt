é possível que a proporção das suas imagens não corresponda ao tamanho que você precisa usar para seguir o layout. O que eu gosto de fazer nesses casos é usar uma propriedade do CSS chamada object-fit, passando o valor cover. Assim, a minha imagem se expande para se adequar ao tamanho definido, mas essa expansão não deixa a imagem distorcida, como geralmente acontece quando você só define a altura e largura. Por exemplo:

.publicacao-destaque img {
   margin-top: .4rem;
   width: 100%;
   height: 152px;
   object-fit: cover;
}
